## Elevator pitch (≈60 words)

GitKosh is a macOS app that mediates between the coding sites you already use and GitHub. Log in once; it pulls every accepted solution, writes a per-problem README with the algorithm and complexity, and commits each one **dated to the day you actually solved it**. It builds a stats dashboard, a portfolio site, Anki/Notion study exports, and an embeddable profile card — automatically.

## Show HN

**Title:** Show HN: GitKosh – Sync coding-practice solutions to GitHub, backdated to solve day

**Body:**
> I kept grinding LeetCode/Codeforces but my GitHub looked dead, and the existing "leetcode-to-github" tools just dump code dated today. GitKosh pulls accepted solutions from six judges, writes a per-problem README (problem → numbered algorithm of *your* code → complexity → key insight) with a pluggable LLM (Gemini/Groq or one-click local Ollama), and commits each solution with the author date set to the **real submission timestamp** via the Git Data API — so the contribution graph reflects when you actually solved things.
>
> It also generates a stats dashboard, a GitHub Pages portfolio, Anki/Notion exports, an embeddable profile stats card, insights + résumé bullets, and a contest tracker. No git/terminal needed; logins use the system WebKit cookie store (no passwords stored); GitHub uses OAuth device flow. macOS app + a beta browser extension for real-time capture (Windows/Linux).
>
> Tech: py2app bundle, PyObjC WKWebView login, GitHub REST/Git Data API. Feedback welcome.
